Beginner's GuideLine

You can click the question mark button at the bottom left anytime to open the tutorial.

Install Extension

1. Please make sure to download theChrome extension,which will mainly capture your tabs and automatically generate draggable cards. (If you are not using Chrome or Edge browser, you will have to add cards manually.)

How to use the Extension?

2. Please make sure you have logged in before clicking the extension icon in the top-left corner. When it shows "ON", it means the detection has started. Next, click "Reload" on the homepage to generate cards for all currently opened tabs.
Note: Please make sure the screen is on the Tobicord website before clicking the extension. Also, please switch the extension to OFF first before closing the website.

Website Button Distribution Map

3. This is a basic introduction to website buttons, and there are more detailed feature introductions coming up.

Basic Process Of Creating And Using Tobicord

4. Simply drag the right-side tab cards to the middle "Collection" section, and they will be saved in your account space and won't be lost in the future.
(Window 1 will be automatically generated only if the extension is installed. For manual addition, please refer to the next article.)

Manually Add Webpage Cards

5.This article introduces how to manually add webpage cards without using the extension, in case you don't have the extension installed.
Note: Webpage cards will not be saved in your account unless you drag and drop them into the Collection area!

Deletion Process

6. This article introduces the deletion process, including tab cards, collections, spaces, and organizations.

Organization Invitation System

7. By entering a member's email address, you can invite different members to join the group and edit the page information together. The invited members can find the invitation notification in the notification bar and decide whether to join, or they can leave the group at any time.

Permission System

8. The creator of the group has the power to adjust the member's permissions, and different roles have their own permissions.
For example, Visitors can only view but not edit any item.
